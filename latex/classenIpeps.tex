\subsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\\*
class en\+Ipeps$<$ T $>$}

A derived class of \hyperlink{classctmBase}{ctm\+Base} describing the environment of 2D i\+P\+E\+PS on square lattice. 

By default the wave function has 2 by 2 unit cell with checkerboard pattern. \hyperlink{classenIpeps}{en\+Ipeps} is used in ipeps to measure physical quantities and update the wave function. Methods defined in \hyperlink{classctmBase}{ctm\+Base} are employee to find the converged environment.
