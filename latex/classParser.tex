\subsection{Detailed Description}
A class to parse input parmeters. 

One can specify the input parameters in input\+File and \hyperlink{classParser}{Parser} will handle the parsing automatically. For example, 
\begin{DoxyCode}
\hyperlink{classParser}{Parser} pars;
\textcolor{keywordtype}{int} dimD;
pars.bind( \textcolor{stringliteral}{"virtual bond dimension"}, dimD );
pars.\hyperlink{classParser_a9df4698487e5b291314e06513d631313}{Parse}( \textcolor{stringliteral}{"inputFile"} );
\end{DoxyCode}
 And in input\+File\+: 
\begin{DoxyCode}
1 virtual bond dimension = 3
\end{DoxyCode}
 will do the job.
