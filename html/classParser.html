Detailed Description

A class to parse input parmeters.

One can specify the input parameters in inputFile and Parser will handle the parsing automatically. For example,

Parser pars;
int dimD;
pars.bind( "virtual bond dimension", dimD );
pars.Parse( "inputFile" );

And in inputFile:

1 virtual bond dimension = 3

will do the job.
