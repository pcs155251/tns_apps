Detailed Description

template<typename T>
class enIpeps< T >

A derived class of ctmBase describing the environment of 2D iPEPS on square lattice.

By default the wave function has 2 by 2 unit cell with checkerboard pattern. enIpeps is used in ipeps to measure physical quantities and update the wave function. Methods defined in ctmBase are employee to find the converged environment.
